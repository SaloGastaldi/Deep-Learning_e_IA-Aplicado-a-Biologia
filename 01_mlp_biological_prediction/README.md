# 🧠 Neural Network Foundations for Biological Prediction

This project introduces the fundamentals of neural networks through a from-scratch implementation of a single neuron using NumPy, applied to simplified biological prediction tasks.

It serves as the conceptual foundation for more advanced deep learning models used in bioinformatics and AI-driven drug discovery.

---

## 🎯 Objectives

- Understand how neural networks learn from data
- Implement forward propagation and gradient descent from scratch
- Analyze decision boundaries in simple biological datasets
- Explore limitations of linear models in classification tasks

---

## 🧬 Biological Context

The problems studied here represent simplified biological prediction scenarios such as:

- Binary classification of biological states
- Relationship modeling between biological variables
- Toy examples of gene expression-like patterns

---

## ⚙️ Technical Approach

- Pure NumPy implementation (no deep learning frameworks)
- Manual forward propagation
- Gradient computation from scratch
- Binary cross-entropy loss
- Batch gradient descent optimization

---

## 🧪 Key Experiments

- Neuron behavior in 1D, 2D, and 3D spaces
- Activation functions comparison (Sigmoid, ReLU, Tanh)
- Logical functions (AND, OR, XOR)
- Training dynamics and convergence analysis

---

## 🚨 Key Insight

A single neuron can only learn linear decision boundaries, making it incapable of solving non-linear problems such as XOR.
